# hr-analytics-dashboard

## Project Scope
This project aims to analyze HR data to understand employee attrition and key workforce indicators.

## Key Questions
- What factors are most correlated with employee attrition?
- Which departments and job roles are the most affected?
- How do satisfaction, salary, and workload impact retention?

## Deliverables
- Exploratory Data Analysis (EDA)
- Cleaned & processed dataset
- Predictive model (attrition)
- HR Dashboard with actionable insights
